# Impossible Ring

Interactive WebGL Möbius strip made of discrete wedge-shaped panels with per-face coloring and edge rendering. Drag to tilt, auto-rotates when released.

**[Live Demo](https://zebiv.com/impossible-ring/)**

## Tech

- Pure WebGL 1 (no libraries)
- Custom GLSL shaders for per-face coloring and edge lines
- Single HTML file, no build step
